## 🛠️ Manual Deployment

```bash
# 1. Clone the repo
git clone https://github.com/mrfrankofcc/SUBZERO-MD
cd SUBZERO-MD

# 2. Install dependencies
npm install

# 3. Configure your environment
#    → Add your SESSION_ID
#    → Edit the .env file

# 4. Start the bot
npm start
```

> ⚠️ Make sure your `SESSION_ID` is set before running — get it at [sessions.subzero.gleeze.com](https://sessions.subzero.gleeze.com)
